# Support

- Use [GitHub Discussions](https://github.com/katooling/decision-deliberation/discussions) for setup help, design questions, and examples.
- Use [bug reports](https://github.com/katooling/decision-deliberation/issues/new?template=bug.yml) for reproducible defects.
- Use [feature requests](https://github.com/katooling/decision-deliberation/issues/new?template=feature.yml) for product needs and proposed improvements.
- Use [private vulnerability reporting](https://github.com/katooling/decision-deliberation/security/advisories/new) for security concerns.

This is a maintainer-led public-preview project. Support is best effort and no response-time SLA is promised.
