# Code of Conduct

## Our standard

We want participation in this project to be safe, constructive, and welcoming. Be respectful, discuss ideas rather than people, give actionable feedback, acknowledge uncertainty, and make room for different levels of experience.

Unacceptable behavior includes harassment, discrimination, threats, sexualized attention, deliberate intimidation, publishing another person's private information, or sustained disruption of project spaces.

## Scope

This standard applies in repository issues, pull requests, Discussions, reviews, and other spaces where someone is representing this project.

## Enforcement

Maintainers may edit or remove comments, close interactions, temporarily restrict participation, or permanently ban participants when behavior harms the community. Enforcement decisions should be proportionate, documented privately, and protect reporters from retaliation.

For a private conduct report, open a [private repository advisory](https://github.com/katooling/decision-deliberation/security/advisories/new) with a title beginning `Conduct:`. This project reuses that private channel so sensitive reports do not have to be posted publicly. For behavior that violates GitHub's platform policies, also use GitHub's Report Abuse flow.
